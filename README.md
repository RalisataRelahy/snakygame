# Snaky GAME🐍✨

Un **Snake** moderne et stylé réalisé avec **Pygame**  
Projet personnel avec un design soigné : glow, dégradé sur le corps, nourriture qui pulse, HUD semi-transparent, menus pause / game over, etc.

https://github.com/RalisataRelahy/snakygame.git

## Fonctionnalités

- Serpent avec **tête lumineuse** + dégradé vert sur le corps
- Nourriture qui **pulse** et a un petit reflet
- Grille de fond subtile
- HUD semi-transparent avec score
- Menu d'accueil stylé
- Écran Game Over avec détection de nouveau record
- **Menu Pause** (touche P ou Échap)
- Contrôles fluides (impossible de faire demi-tour direct)
- Meilleur score conservé pendant la session

## Contrôles

| Touche          | Action                          |
|-----------------|---------------------------------|
| ← ↑ → ↓         | Déplacer le serpent             |
| **P** ou **Échap** | Mettre en pause / Reprendre   |
| **Espace**      | Reprendre (depuis pause)        |
| **R**           | Recommencer la partie           |
| **M** ou **Q**  | Retour au menu principal        |
| **Espace**      | Lancer une partie (menu)        |

## Technologies

- Python 3
- Pygame
